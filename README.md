# Animais Fantásticos

A responsive and interactive web project built entirely with **Vanilla JavaScript (ES6+)**, HTML5, and CSS3. This repository was created as part of a comprehensive JavaScript ES6+ course to practice and master modern web development concepts, DOM manipulation, and UI interactions.

## 🚀 Live Access

Check out the live application here: **[Animais Fantásticos](https://jpsantosss.github.io/html-animais-fantasticos/)**

## 🛠️ Technologies

- **HTML5** - Semantic structure
- **CSS3** - Styling, Flexbox, CSS Grid, and custom animations
- **JavaScript (ES6+)** - Vanilla JS, Modules, Classes, Promises, Fetch API, and DOM manipulation

## ✨ Key Features

- **Interactive UI Components:** Includes Tab Navigation, Accordion Lists, Modals, Tooltips, and Dropdown Menus.
- **Smooth Navigation:** Smooth scrolling for internal links and dynamic scroll animations.
- **Responsive Design:** Fully adapted for both desktop and mobile devices.
- **Modern Architecture:** Clean, modularized JavaScript code for better maintainability.

## 💻 Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jpsantosss/html-animais-fantasticos.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd html-animais-fantasticos
   ```

3. **Run the application:**
   Simply open the index.html file in your preferred web browser. No complex build tools or dependencies are required. (Tip: You can use the "Live Server" extension in VS Code for a better development experience).
